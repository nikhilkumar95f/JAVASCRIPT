# JavaScript Learning Repository

This repository contains beginner-friendly JavaScript practice files and examples organized by topic.

## Contents

- 01_basics: Variables, data types, type conversion, comparisons, strings, numbers, math, dates, and basic practice exercises
- 02_basics: Arrays and objects
- 03_basics: Functions, scopes, arrow functions, and lifecycle-related examples

## How to Use

1. Open any JavaScript file in the folders above.
2. Run the file using Node.js:
   ```bash
   node filename.js
   ```
3. Practice by editing the examples and experimenting with the code.

## Goal

This repo is intended for learning and revising core JavaScript concepts step by step.
